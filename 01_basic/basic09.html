変数と代入

変数は後で使いたい数字や、文字、データなどを保存できるものです。
演算子と変数を両方使ってみましょう。

変数を作成し、出力する2

コンソール画面に下記の順序で内容を作成し、出力してみましょう
・変数 name を作成する
・変数 name に[自分の名前]を代入する
・コンソール画面に「私の名前は [自分の名前] です。」と出力する

Point

文字と変数を組み合わせて表示したい場合には、結合するための演算子が必要です。
文字列の中に変数を組み込みたい場合バッククォートを使った方法も可能です。
変数である目印のために ${変数名} として記入する必要がありますが、わざわざクォーテーションを途中で切る必要がないため、合わせて覚えておくと便利です。
例
`Hello ${name}` // バッククォートは Shiftキー + @ で表示されます（JISキーボードの場合）
